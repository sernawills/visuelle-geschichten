Biodiversity

For my next project, I decided to use a JSON file to create data visualizations. I was interested in exploring forest coverage and used two data sets to create my visualizaton. The first data set included the percentage of forest coverage for each country, while the second data set provided the coordinates for each country. To make the data more accessible, I merged the two data sets so that each country was assigned to a set of coordinates. My goal was to create an abstract map that would allow viewers to visualize forest loss over the years. To achieve this, I decided to use circles without fill to represent each country. The filling within the circles would then represent the forest coverage. I also made the visualization interactive, so that viewers could see the changes for themselves. After I completed the project, I was surprised by some of the findings. One of the most interesting things I noticed was that the forest coverage had been increasing in the global north while decreasing in the global south. I also experimented with adding Perlin noise to the circles to make the interaction smoother, but it didn't have the effect I was looking for. Overall, this project was a great learning experience. I gained a better understanding of data visualization and was able to convey a powerful message about forest coverage and global trends.
